# ForYourStomach
TopDown Shooting game with Unreal Engine4(Blueprints)
Started at 17.11.22

Overview
--------
Save your Stomach. You can use various bullet and try to kill germs.
WASD: Moving
Left Mouse Click: Shoot
Right Mouse Click: Teleport
Q, E: Changing bullet
M: Pause menu
